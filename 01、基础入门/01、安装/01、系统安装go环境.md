

# 系统安装go环境

## PKG 安装方式

MAC 直接使用 pkg 来安装
<https://golang.org/dl/>

安装地址在
> /usr/local/go

配置GO 环境

```shell script
vim  ~/.bash_profile
```

添加下面配置

```shell script
# goland path
export GOROOT=/usr/local/go
export PATH=$PATH:$GOROOT/bin
```

执行生效
```shell script
source ～/.bash_profile
```
## DOCKER 安装 go

```shell script
docker pull golang
```
